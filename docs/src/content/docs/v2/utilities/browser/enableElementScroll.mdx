The `enableElementScroll` function is used to re-enable element scroll after disabling it with <a href={getUtilityUrl('browser', 'disableelementscroll')}>disableElementScroll</a> utility

## Usage
```ts
import { enableElementScroll } from '@mustib/utils/browser'

enableElementScroll()
```

## Definition
```ts
function enableElementScroll(options?: {
  element?: HTMLElement
}) { }
```
